# Ranju Hasan Khan - Portfolio

A modern, beautiful portfolio website showcasing my work as a Full-Stack Developer and Peace Advocate.

## Overview nice

This portfolio features:
- **Modern Design**: Dark theme with sleek animations
- **Fully Responsive**: Works perfectly on all devices
- **Interactive Elements**: Custom cursor, smooth scrolling, intersection observers
- **Performance Optimized**: Fast loading times

## Tech Stack

- HTML5
- CSS3 (with modern layouts and animations)
- Vanilla JavaScript (no frameworks)
- Deployed on Vercel

## Features

- **Hero Section**: Eye-catching introduction with typing animation
- **About**: Background and statistics
- **Skills**: Core capabilities in 6 categories
- **Experience**: Timeline of work and education
- **Tech Stack**: Technologies used (MERN stack)
- **Contact**: Multiple ways to get in touch
- **Custom Cursor**: Interactive mouse tracking

## Getting Started

### Development

```bash
npm install
npm run dev
```

This starts a local server at http://localhost:8080

### Build & Deploy

The site is automatically deployed on Vercel. Simply push to your repository:

```bash
git push origin main
```

Vercel will automatically build and deploy the latest changes.
